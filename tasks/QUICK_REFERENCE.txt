-----------

bg-gray-100                 -> bg-muted

bg-gray-500                 -> bg-secondary

text-gray-600               -> text-muted-foreground
text-gray-700               -> text-foreground/70
text-gray-800               -> text-foreground
text-gray-900               -> text-foreground

border-gray-50              -> border-border
border-gray-100             -> border-border
border-gray-200             -> border-border

border-gray-400             -> border-border
border-gray-500             -> border-secondary

YELLOW/AMBER COLORS (WARNING)
-----------------------------
text-yellow-400             -> text-warning (or text-amber-400)

SLATE COLORS
------------
bg-slate-800                -> bg-muted or bg-card
bg-slate-900                -> bg-muted or bg-background (dark mode)

border-slate-600            -> border-border
border-slate-700            -> border-border


INLINE HEX COLORS TO CSS VARIABLES
-----------------------------------
#ffffff                     -> var(--foreground) or var(--primary-foreground)

#1f2937 (gray-800)          -> var(--muted)
#374151 (gray-700)          -> var(--border)
#94a3b8 (slate-400)         -> var(--muted-foreground)
#3b82f6 (blue-500)          -> var(--primary)
#10b981 (emerald-500)       -> var(--success)
#ef4444 (red-500)           -> var(--destructive)
#EF4444 (red-500)           -> var(--destructive)


FOCUS STATES
------------
focus:ring-blue-500         -> focus:ring-primary
focus:border-blue-500       -> focus:border-primary
focus:ring-2                -> (keep as is)
focus:outline-none          -> (keep as is)


HOVER STATES
------------
hover:bg-gray-50            -> hover:bg-muted
hover:bg-blue-700           -> hover:bg-primary/90
hover:text-gray-900         -> hover:text-foreground
hover:text-blue-600         -> hover:text-primary
hover:border-gray-300       -> hover:border-border
hover:border-blue-500       -> hover:border-primary


OPACITY VARIANTS
----------------
bg-primary/10               -> (keep as is, but ensure primary is semantic)
bg-primary/20               -> (keep as is)
text-foreground/80          -> (keep as is)
text-foreground/70          -> (keep as is)

For hardcoded: Create from CSS variables
Example: #3b82f6 with 10% opacity -> rgba(59, 130, 246, 0.1)

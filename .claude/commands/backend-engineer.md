Use the Backend Engineer agent for API design, database performance, and backend architecture reviews.

Specify the code/API to review, and the agent will:
1. Review API design and contracts
2. Analyze database queries and indexing
3. Check error handling and reliability
4. Verify security practices
5. Suggest performance optimizations

Example usage:
```
/backend-engineer

Review the SMS sending service for performance bottlenecks and database query optimization.
```

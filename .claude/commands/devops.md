Use the DevOps agent for infrastructure decisions, deployment planning, and scaling strategies.

Specify your infrastructure challenge, and the agent will:
1. Design deployment pipelines
2. Plan scaling strategies
3. Configure monitoring and alerting
4. Create disaster recovery plans
5. Recommend infrastructure improvements

Example usage:
```
/devops

Design a deployment pipeline and monitoring strategy for our production Render deployment.
```

Use the QA Testing agent to plan tests, create test cases, and validate feature quality.

The QA Testing agent provides:
- Comprehensive test plan creation
- Test case design with edge cases and error scenarios
- Acceptance criteria definition
- Testing strategy and automation recommendations
- Quality risk identification
- Test data preparation templates
- Bug report and severity classification standards
- Regression test planning

**Examples of good questions:**
- "Create a test plan for the 10DLC delivery tier feature including all edge cases"
- "What test cases should we include for SMS sending with 10x growth?"
- "Design a comprehensive test strategy for member import functionality"
- "Create acceptance criteria for the onboarding wizard feature"
- "What are the critical paths we should test for production deployment?"

Simply ask your testing question and the QA Testing agent will provide detailed test plans with specific test cases, preconditions, and acceptance criteria.
